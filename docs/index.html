About This Demo

Production-ready agentic AI system built with RAG architecture, tool calling, and hallucination prevention.

Tech Stack: FastAPI + OpenAI GPT-4o-mini + FAISS + 8 agentic tools. Deployed on Render.com with auto-scaling.

Agentic AI System

  • Tool Calling: 8 specialized tools (search, filter, recommend, hours, contact...)
  • Multi-step Planning: Decomposes complex queries → executes tools in parallel
  • Context Aggregation: Combines results from multiple tool executions
  • Conversational Memory: Tracks last 10 exchanges for follow-up questions
  • Auto-Validation: 4-type hallucination prevention (restaurants/schedules/prices/departments)

RAG Architecture

  • FAISS Semantic Search: Vector similarity on 20 restaurants + 32 menu items
  • Zero Hardcoded Data: Single source of truth (knowledge base)
  • Automated CI/CD Pipeline: Weekly web scraping + embeddings rebuild
  • Response Time: <500ms end-to-end (embedding + retrieval + LLM)

Production Features

  • Multilingual: Auto-detects and responds in query language
  • Clickable Links: Preserved HTML from scraped content
  • Markdown-Free: Plain text responses (no formatting artifacts)
  • Deployment: Render.com auto-scaling, production-grade
