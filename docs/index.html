About This Demo

Production-ready agentic AI system built with RAG architecture, tool calling, and hallucination prevention.

Tech Stack: FastAPI + OpenAI GPT-4o-mini + FAISS + 8 agentic tools. Deployed on Render.com with auto-scaling.

Agentic Capabilities

  • Multi-step reasoning & planning
  • 8 specialized tools (search, filter, recommend)
  • Context aggregation from multiple sources
  • 4-type hallucination validator
  • Semantic search via FAISS (20 restaurants, 32 menu items)

Key Features

  • 100% RAG architecture (zero hardcoded data)
  • Multilingual (French/Vietnamese/English)
  • Real-time responses (<500ms)
  • Weekly auto-refresh (GitHub Actions)
  • Production validation pipeline
